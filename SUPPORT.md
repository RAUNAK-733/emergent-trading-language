# Support

Use GitHub issues for reproducible project problems and feature proposals.

Before opening an issue:

1. read `README.md` and the files in `docs/`;
2. run `python main.py status`;
3. run the test suite;
4. include the Python, PyTorch, CUDA, and operating-system versions;
5. include the exact command and relevant error output.

This repository is a research project. General reinforcement-learning tutoring,
unrelated environment setup, and private checkpoint analysis may not receive
support.
